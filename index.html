what is html?

HTML stands for HyperText Markup Language. It's a programming language that defines the structure of web pages. HTML is the standard markup language for documents that are displayed in a web browser.

what is css?

Cascading Style Sheets (CSS) is a programming language that describes the appearance and formatting of a document written in markup language. It is used to lay out and structure web pages.

what is javascript?

JavaScript (JS) is a text-based programming language used both on the client-side and server-side that allows you to make web pages interactive. It is one of the three core technologies of the World Wide Web, along with HTML and CSS.

what is frameworks?

A framework is a structure that can be used to support or guide the building of something. It can also refer to the basic structure, arrangement, or system.
